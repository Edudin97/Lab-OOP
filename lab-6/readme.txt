Написать программу, отыскивающую проход по лабиринту, с использованием контейнерного класса stack из STL.
Лабиринт представляется в виде матрицы, состоящей из квадратов. Каждый квадрат либо открыт, либо закрыт.
Вход в закрытый квадрат запрещен. Если квадрат открыт, то вход в него возможен со стороны, но не с угла. 
Программа находит проход через лабиринт, двигаясь от заданного входа.
После отыскания прохода программа выводит найденный путь в виде координат квадратов.
